➕ Adding a new shirt
Step 1 — Create a folder inside shirts/ and drop your PNGs in it:
shirts/my-new-shirt/   ← any PNGs, any names

Step 2 — Open index.html, find the PRODUCTS list, and paste this block before the closing ];
js{
  id:     "my-new-shirt",    // ← must exactly match the folder name
  series: "pixel",           // ← must match a key in SERIES
  name:   "My New Shirt",    // ← what shows on the card
  price:  "$34.00",
  desc:   "Short description of the shirt.",
  url:    "https://www.printful.com/...",   // ← your Printful buy link
},


➕ Adding a new series (e.g. Galaxy Series)

Step 1 — In index.html, find the SERIES block and paste inside it:
js"galaxy": {
  label: "Galaxy Series",
  desc:  "Cosmic drops. Deep space aesthetics on premium cotton.",
},

Step 2 — On any products that belong to that series, set:
jsseries: "galaxy",
```

A new tab appears in the store automatically. No other changes needed.
